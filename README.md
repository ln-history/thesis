# Thesis

This repository contains the raw latex files used to compile my Bachelor-Thesis:
`Catching Lightning in a Database: An Architecture for Real-Time Gossip Preservation`

The [./analysis](./analysis) directory contains a python project that was used
to create the figures in the latex project.
This repo exists to make the research reproducable and open-source.

The final compiled thesis (with removed personal information),
can be found at [./main.pdf](./main.pdf).
